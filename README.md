# Moran Racing Shop - E-Commerce

Tienda en línea para productos de SimRacing y accesorios de Fórmula 1.

## Características

- Catálogo de productos
- Formulario de compra
- Página de contacto
- Responsive design
- Accesibilidad ARIA

## Estructura del Proyecto

```
├── index.html                 # Página principal
├── productos.html             # Página de productos
├── formulario.html            # Formulario de compra
├── contacto.html              # Página de contacto
├── confirmacion.html          # Página de confirmación
├── assets/
│   ├── styles/
│   │   └── styles.css         # Estilos principales
│   └── [imágenes de productos]
└── README.md
```

## Cómo usar

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. Navega por el sitio usando los enlaces de navegación

## Despliegue en GitHub Pages

Este proyecto está configurado para ser desplegado en GitHub Pages. Las rutas están configuradas como rutas relativas, lo que permite que funcione tanto localmente como en GitHub Pages.

### Pasos para desplegar:

1. Sube los archivos a tu repositorio de GitHub
2. Ve a Settings > Pages
3. Selecciona la rama `main` como fuente
4. Tu sitio estará disponible en: `https://tu-usuario.github.io/E-Commers/`
